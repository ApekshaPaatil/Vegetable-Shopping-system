# Vegetable Shopping System

## Project Description
Online Vegetable Shopping System is a web application that allows customers to browse vegetables, add items to a cart, and place orders online. The system is developed using Python Flask, HTML, CSS, Bootstrap and MySQL.

## Features
- User registration and login
- Browse vegetables
- Add products to cart
- Place orders
- Admin dashboard
- Manage products and categories

## Technologies Used
- Python (Flask)
- HTML
- CSS
- Bootstrap
- MySQL

## Installation
1. Clone the repository
2. Install required packages
3. Run the Flask application
